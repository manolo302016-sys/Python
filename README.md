# Enterprise Data Science Project

This project provides a highly structured Python environment for automated data science tasks, from ingestion to predictive visualization.

## Setup

1. Enable the virtual environment:
   `.\venv\Scripts\Activate.ps1`
2. Install dependencies:
   `pip install -r requirements.txt`

## Structure

* `data/`: Raw and processed datasets.
* `docs/`: Documentation.
* `notebooks/`: Jupyter notebooks for exploratory analysis.
* `src/`: Core Python modules for the data pipeline.
* `tests/`: Automated test suite.

## Execution
Run JupyterLab:
`jupyter lab`
